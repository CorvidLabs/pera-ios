# PeraWallet_Classes_Messaging — Requirements

## User Stories

- As a user, I can open a conversation list and tap into a chat thread.
- As a user, I can type and send a message from the thread.

## Acceptance Criteria

- [ ] The feature is reachable only when `enable_messaging` is on.
- [ ] Errors surface as a user-facing alert.
- [ ] The UI never imports the AlgoChat SDK directly.
